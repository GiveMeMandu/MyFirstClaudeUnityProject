## Required Updates

### 1. 활성 태스크 문서 업데이트
`Docs/dev-docs/active/` 내 각 태스크에 대해:

- `[task-name]-context.md` 업데이트:
  - 현재 구현 상태
  - 이번 세션에서 내린 핵심 결정사항
  - 수정된 파일 및 이유
  - 발견된 블로커 또는 이슈
  - 다음 즉시 수행할 단계
  - Last Updated 타임스탬프

- `[task-name]-tasks.md` 업데이트:
  - 완료된 태스크를 ✅ 로 표시
  - 새로 발견된 태스크 추가
  - 진행 중인 태스크에 현재 상태 업데이트
  - 필요 시 우선순위 재정렬

### 2. 세션 컨텍스트 캡처
다음에 관련된 정보 포함:
- 해결한 복잡한 문제
- 내린 아키텍처 결정
- 발견하고 수정한 버그
- 발견한 통합 지점
- 사용한 Unity 관련 패턴 또는 접근 방식

### 3. 미완성 작업 문서화
- 컨텍스트 한계에 도달했을 때 작업 중이던 내용
- 부분적으로 완료된 기능의 정확한 상태
- 재시작 시 실행해야 할 명령어 (unity-cli 등)
- 영구적 수정이 필요한 임시 해결책

### 4. 핸드오프 노트 작성

- 편집 중이던 정확한 파일 및 줄

- 주의가 필요한 미커밋 변경사항
- 작업 검증을 위한 테스트 명령어
