- **작성일**: 2026-03-26

- **slug**: resource-system

---

## 1. 개요

### 목적
기지 경제의 핵심. 3종 자원(기초/고급/방어)을 관리하고, 건설/업그레이드/방어에 소비하여 성장과 방어 사이의 템포 전략을 결정하는 시스템.

### 핵심 경험
- **템포의 싸움**: 방어에 투자하면 안전하지만 성장이 느리고, 성장에 투자하면 스노우볼이 굴러가지만 다음 방어전이 위험
- **간결한 경제**: 자원 3종으로 복잡도를 낮추되, 항상 선택을 강제
- **생산의 보람**: 턴 종료 시 자원이 증가하는 피드백 → "한 턴만 더" 동기

## 2. 시스템 설계

### 핵심 메커니즘

#### 2.1 자원 종류

| 자원 | 역할 | 생산 수단 | 비유 |
|---|---|---|---|
| 기초 자원 | 건설/업그레이드 비용, 범용 | 자원 건물(Farm 등) + 전투 보상 | 스타크래프트 미네랄 |
| 고급 자원 | 건설/업그레이드 비용(고급) | 자원 건물(Mine 등) + 탐사 발견 | 스타크래프트 가스 |
| 방어 자원 | 방어 건물 운영 비용 | 특수 건물(소량) + 탐사 발견 | 탄약 |

- 무한 축적 가능 (창고 시스템 없음)
- 최소값 0 (음수 불가)

#### 2.2 자원 생산
- 자원 건물에 인력을 배치하면 **턴 종료 시(밤 전환 전)** 자원 생산
- 건물 종류에 따라 생산하는 자원 종류가 다름
- 자원 건물도 **다중 슬롯** 보유 (인력 배치 시스템과 연동)
- 슬롯별 생산량은 `effectPerWorker` 값으로 결정

#### 2.3 자원 소비
| 소비 시점 | 소비 내용 | 조건 |
|---|---|---|
| 건설 명령 시 | 기초 + 고급 자원 | 보유량 ≥ 비용 |
| 업그레이드 명령 시 | 기초 + 고급 자원 | 보유량 ≥ 비용 |
| 전투 시작 시 | 방어 자원 | 배치된 방어 인력 슬롯 수 × 슬롯당 소모량 |

#### 2.4 방어 자원 검증
- 턴 종료 시 방어 건물에 배치된 인력의 총 방어 자원 소모량을 계산
- **보유 방어 자원 < 소모 예정량** → 턴 종료 차단 + "방어 자원 부족 — 방어 인력 조정 바람" 토스트
- 플레이어가 방어 인력을 줄이거나 방어 자원을 확보한 후 재시도

#### 2.5 전투 보상
- 전투 승리 시 자원 보상 (기초/고급 자원)
- 보상량은 기획자가 WaveDataSO 또는 별도 SO에서 설정

### 데이터 흐름

```
[입력]                              [처리]                           [출력]
─────────────────                 ─────────────────               ─────────────────
인력 배치 (자원 건물)    →     ResourceManager           →     턴 종료 시 자원 생산
건설/업그레이드 명령     →     (비용 검증 + 차감)         →     자원 차감 or 부족 경고
턴 종료 트리거          →     (방어 자원 검증)           →     통과 or 차단 + 토스트
전투 시작              →     (방어 자원 차감)           →     방어 자원 감소
전투 승리              →     (보상 지급)               →     자원 증가
```

## 4. 밸런스 가이드

### 조정 가능 파라미터

| 파라미터 | 조정 목적 | 권장 범위 |
|---|---|---|
| 초기 기초 자원 | 시작 여유도 | 80~150 |
| 초기 고급 자원 | 초반 업그레이드 가능 횟수 | 20~50 |
| 초기 방어 자원 | 첫 전투 방어 여유 | 15~30 |
| 자원 건물 슬롯 effectPerWorker | 생산 효율 | 5~15/턴 |
| 방어 슬롯당 소모량 | 방어 운영 비용 | 1 (기본) |
| 건설 비용 (건물별) | 건설 진입 문턱 | SO에서 개별 설정 |
| 전투 보상 | 방어 동기 | SO에서 설정 |

### 밸런스 기준

- **3턴 내 건설 1개**: 초기 자원으로 즉시 건설 가능, 하지만 2개는 생산 후에야 가능
- **방어 압박**: 방어 건물 2개 풀가동 시 방어 자원이 3~4턴 내 고갈 → 생산 투자 필요
- **고급 자원 희소**: 고급 자원이 필요한 건설/업그레이드는 신중한 선택 필요
- **템포 보상**: 방어 투자를 줄이고 성장에 투자하면 2~3턴 후 생산량 증가로 역전 가능

## 5. 엣지 케이스

| 상황 | 처리 방법 |
|---|---|
| 자원 부족 시 건설 시도 | 버튼 색 변경 + "자원 부족" 토스트 메시지 |
| 방어 자원 부족으로 턴 종료 | 턴 종료 차단 + "방어 자원 부족 — 방어 인력 조정 바람" 토스트 |
| 자원이 0 미만으로 떨어지려는 경우 | 0으로 클램프 (음수 불가) |
| 자원 건물 파괴 후 턴 종료 | 이미 턴 종료 시 생산 완료 후 밤 진입이므로 영향 없음 |
| 전투 없는 밤에 방어 자원 | 차감하지 않음 (전투 시에만 소모) |
| 모든 자원 0인 상태 | 건설/업그레이드 불가, 방어 인력 배치 불가, 자원 건물 인력 배치로 생산 필요 |
