# V2 기능 확인 인터뷰 체크리스트

Play 모드에서 확인하면서, 기획서 대비 현재 구현 상태를 점검합니다.
**씬**: `Assets/Scenes/V2_Game.unity` (또는 Window > Project Sun > Build Game Scene)

---

## 1. 메인 메뉴 (SF-UX-001/002)
- [ ] NEW GAME → 기지 선택 화면 전환
- [ ] 기지 선택 → START → 게임 시작
- [ ] CONTINUE 버튼: 세이브 없으면 비활성화
- [ ] QUIT 동작 여부

**기획서**: Vision.md — 진입 장벽 최소화. 즉시 게임 진입 가능해야 함.

---

## 2. 낮 페이즈 — 탭 전환
- [ ] Construction / Workforce / Exploration 탭 버튼 클릭 반응
- [ ] 자원 표시 (Basic/Advanced/Relic) 정확성
- [ ] TURN 배지 표시

**기획서**: turn-system.md — 낮 페이즈에서 건설/인력/탐사 의사결정.

---

## 3. 건설 탭 (SF-CON-009)
- [ ] 슬롯 리스트: 8개 슬롯 (활성 4 + 잠김 4) 표시
- [ ] 슬롯 클릭 → 상세 패널 (이름/HP/스탯)
- [ ] 잠김 슬롯 시각적 구분 (opacity 낮음)
- [ ] 손상 건물: 빨간 테두리 + REPAIR 버튼

**기획서**: construction-system.md
- 고정 슬롯 시스템 (Thronefall 패턴)
- 건설 비용 자원 감소
- 업그레이드 분기 A/B 선택 (비가역)

**확인 질문**:
- Q: 현재 건설 비용이 차감되는가?
- Q: 업그레이드 분기 카드가 표시되는가?
- Q: 건물 SO 에셋 없이도 기본 정보가 보이는가?

---

## 4. 관리 탭 (SF-WF-009)
- [ ] 시민 4명 리스트 표시
- [ ] 필터 (All/Idle/Assigned/Injured) 작동
- [ ] 시민 클릭 → 상세 패널 (이름/적성/레벨/상태)
- [ ] Assign to Combat → 상태 InCombat 전환
- [ ] Unassign → Idle 복귀

**기획서**: workforce-system.md
- "디아블로 보석 소켓" 모델: 시민을 건물 슬롯에 배치
- 5~15명 소수 정예
- 적성별 보너스 (전투/건설/연구/탐사)

**확인 질문**:
- Q: 적성별 포트레잇 색상 구분이 되는가?
- Q: 부상 시민이 빨간 표시로 보이는가?
- Q: 전투 배치된 시민이 밤 전투에 분대로 등장하는가?

---

## 5. 탐험 탭 (SF-EXP-005)
- [ ] 노드맵: 기지(HQ) + 20개 노드 표시
- [ ] 노드 클릭 → 선택 테두리
- [ ] DISPATCH EXPEDITION → 시민 OnExpedition 전환
- [ ] INVEST BONFIRE 버튼 → 자원 차감 + 확률 시민 합류
- [ ] 활성 원정 목록 표시

**기획서**: exploration-system.md
- 노드 그래프 탐사
- 원정대 2~3턴 이동
- 자원/시민/정찰 보상

**확인 질문**:
- Q: 파견 후 2턴 뒤에 보상 로그가 출력되는가?
- Q: 방문한 노드가 Visited 색상으로 바뀌는가?
- Q: 모닥불 투자 시 자원이 차감되는가?

---

## 6. 밤 전환 흐름
- [ ] START NIGHT → 낮 UI 숨김 + 웨이브 미리보기 오버레이
- [ ] 미리보기: 4방향 적 인디케이터 + 적 수/웨이브 수
- [ ] CLOSE → 전투 시작 (ECS 활성화)
- [ ] 전투 HUD: 웨이브 바, 적 카운트, 분대 HP, 건물 HP, 배속 버튼

**기획서**: dots-defense-system.md
- 밤 = ECS 자동 전투 (가벼운 RTS 마이크로)
- 1x/2x 배속 + 일시정지
- 본부 HP 0 = 게임오버

**확인 질문**:
- Q: ECS 적이 실제로 스폰되고 이동하는가?
- Q: 타워가 적을 자동 공격하는가?
- Q: 배속 버튼(1x/2x/일시정지)이 작동하는가?
- Q: 분대가 전투에 참여하는가?
