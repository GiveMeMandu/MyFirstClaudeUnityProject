## 2. 전체 네비게이션 맵

```
[메인 메뉴]
    |
    +-- [새 게임 / 계속하기]
    |
    v
[기지 선택 화면]
    |
    v
[낮 페이즈 — 메인 허브]
    |
    +-- [건설 탭] ----------- 슬롯 클릭 -> [건설/업그레이드 패널] (오버레이)
    |                                         분기 선택 -> [분기 비교 팝업]
    |
    +-- [관리 탭] ----------- 시민 카드 클릭 -> [시민 상세 팝업]
    |   (2일차~ 해금)          시민 드래그/클릭 -> 소켓/분대/미배치 재배치
    |                          분대 구성 버튼 -> [분대 편성 패널]
    |
    +-- [탐험 탭] ----------- 노드 클릭 -> [노드 정보 패널]
    |   (탐험 해금 후)          파견 버튼 -> [원정대 편성 팝업]
    |                          원정대 귀환 -> [귀환 보고 팝업] (자동 발생)
    |
    +-- [연구/정책 버튼] ----- [연구 트리 오버레이]
    |
    v
[낮 종료 버튼]
    |
    v
[웨이브 미리보기 화면]  (전용 화면, 모달 풀스크린)
    |
    v
[전투 준비 상태]  (일시정지 상태의 밤 페이즈, 분대 초기 배치)
    |
    v
[밤 페이즈 — 전투 HUD]
    |
    +-- Space: 일시정지 토글
    +-- 1~5: 분대 선택
    +-- 우클릭(빈 지점): 이동 명령
    +-- 우클릭(적): 공격 명령
    +-- H / 정지 버튼: 정지 명령
    +-- 배속: 1x / 2x 토글
    |
    v
[전투 결과 화면]  (전용 화면)
    |
    v
[낮 페이즈 — 메인 허브]  (다음 턴)
    (귀환 보고가 있으면 자동 팝업)
```

## 3. 화면 목록

| 화면 ID | 화면명 | 유형 | 접근 방법 | 핵심 기능 |
|---|---|---|---|---|
| SCR-01 | 메인 메뉴 | 전용 풀스크린 | 게임 시작 | 새 게임 / 계속하기 / 설정 / 종료 |
| SCR-02 | 기지 선택 | 전용 풀스크린 | 새 게임 시 | 기지 목록 + 기지별 설명 + 선택 |
| SCR-03 | 낮 페이즈 허브 | 탭 허브 (영구) | 밤 전투 종료 후 | 건설 / 관리 / 탐험 탭 전환 |
| SCR-03A | 건설 탭 | 허브 탭 | 탭 클릭 | 기지 뷰 + 슬롯 상태 표시 |
| SCR-03B | 관리 탭 | 허브 탭 | 탭 클릭 (2일차~) | 시민 목록 + 소켓 배치 + 분대 편성 |
| SCR-03C | 탐험 탭 | 허브 탭 | 탭 클릭 (해금 후) | 탐사 노드 맵 + 원정대 현황 |
| PNL-01 | 건설/업그레이드 패널 | 오버레이 패널 | 슬롯 클릭 | 건물 정보 + 건설/업그레이드 실행 |
| PNL-02 | 분기 비교 팝업 | 모달 팝업 | 업그레이드 분기 진입 시 | A/B 분기 효과 나란히 비교 |
| PNL-03 | 시민 상세 팝업 | 모달 팝업 | 시민 카드 클릭 | 시민 스탯 / 숙련도 / 패시브 |
| PNL-04 | 원정대 편성 팝업 | 모달 팝업 | 탐험 탭 파견 버튼 | 시민 선택 + 영향 경고 + 파견 확인 |
| PNL-05 | 귀환 보고 팝업 | 모달 팝업 | 원정대 귀환 턴 자동 발생 | 보상 / 발견 / 인카운터 결과 |
| PNL-06 | 연구 트리 오버레이 | 오버레이 패널 | 연구 버튼 | DAG 기술 트리 + 현재 진행 노드 |
| PNL-07 | 인카운터 선택지 팝업 | 모달 팝업 (블로킹) | 탐사 중 인카운터 발생 | 3선택지 제시 + 선택 실행 |
| SCR-04 | 웨이브 미리보기 | 전용 풀스크린 | 낮 종료 버튼 | 적 진입 방향 / 규모 / 구성 확인 |
| SCR-05 | 밤 페이즈 | 영구 HUD | 웨이브 미리보기 확인 | 전투 진행 + 분대 조작 + 배속/일시정지 |
| SCR-06 | 전투 결과 | 전용 풀스크린 | 전투 종료 자동 전환 | 판정 + 피해 요약 + 보상 목록 |
| SCR-07 | 게임 오버 | 전용 풀스크린 | 본부 파괴 시 | 결과 통계 + 재시도 / 메뉴 복귀 |

## 4. 페이즈 전환 흐름

### 4.1 낮 → 밤 전환

```
낮 페이즈 허브
    |
    플레이어: [낮 종료] 버튼 클릭
    (버튼 위치: 화면 우측 하단 고정)
    |
    v
웨이브 미리보기 화면 (SCR-04)
    - 적 진입 방향 화살표 표시
    - 규모 텍스트 (소/중/대)
    - 탐사 수준에 따른 상세 정보
    - [전투 준비] 버튼
    |
    v
전투 준비 상태 (일시정지 잠금)
    - 밤 페이즈 HUD 진입, 자동 일시정지
    - 분대 초기 배치 가능
    - [전투 시작] 버튼으로 실시간 진행 시작
    |
    v
밤 페이즈 (SCR-05)
```

**설계 의도**: 낮 종료 후 곧바로 전투가 시작되지 않는다. 웨이브 미리보기와 전투 준비 단계가 두 단계 완충재 역할을 한다. 플레이어가 정보를 확인하고 분대를 배치할 시간을 보장한다.

### 4.2 밤 → 낮 전환

```
밤 페이즈 — 마지막 웨이브 격퇴
    |
    v
전투 결과 화면 (SCR-06)
    - 판정 배너 (0.5초 지연 후 등장)
    - 피해 요약
    - 보상 목록
    - [계속] 버튼
    |
    v
낮 페이즈 허브 (SCR-03) — 다음 턴
    - 귀환 원정대가 있으면 [귀환 보고 팝업] 자동 발생
    - 새로운 턴 자원 수입 반영됨
```

**설계 의도**: 전투 결과 화면은 중간 숨 고르기 지점이다. 플레이어가 이번 밤의 결과를 확인하고, 다음 낮의 전략(수리 우선순위 등)을 생각할 시간을 제공한다. 결과 화면을 건너뛰는 옵션은 제공하지 않는다.

## 6. 모달 우선순위 규칙

여러 팝업이 동시에 발생할 수 있을 때의 처리 순서:

| 우선순위 | 팝업 | 처리 방식 |
|---|---|---|
| 1 (최우선) | 인카운터 선택지 (PNL-07) | 블로킹 — 선택 전까지 다른 조작 불가 |
| 2 | 귀환 보고 (PNL-05) | 낮 페이즈 진입 직후 자동 표시, X로 닫기 가능 |
| 3 | 건물 손상 경고 (밤) | 비블로킹 알림 배너 (화면 상단, 3초 자동 소멸) |
| 4 | 탭 해금 알림 | 비블로킹 알림 배너 (화면 상단, 5초 자동 소멸) |

## 8. 정보 공개 레벨 (웨이브 미리보기)

Vision.md와 WaveDefense.md의 탐사 연동 정보 공개 정책.

| 수준 | 조건 | 표시 내용 | UX 표현 |
|---|---|---|---|
| 기본 | 항상 (탐사 없이도) | 적 진입 방향 화살표 + "소/중/대" 텍스트 | 화살표 + 글자 |
| 상세 | 탐사 Lv1+ 또는 감시탑 Lv2+ | 기본 + 적 아키타입 아이콘 (Tier 1/2/3) | 아이콘 그룹 추가 |
| 정밀 | 탐사 Lv2+ 또는 감시탑 Lv3 | 상세 + 적 종류별 수량 + 추가 경로 표시 | 수량 숫자 + 추가 화살표 |

**UX 설계 의도**: 정보가 없는 상태(기본)도 플레이 가능하다. 탐사는 "더 잘 대비할 수 있는 정보"를 주는 보상이며, 탐사를 안 해도 게임이 막히지 않는다. Rising Demacia의 사전 정보 공개 원칙을 부분 적용한 것으로, 기본 수준은 항상 공개하되 상세 정보는 탐사 투자의 보상으로 제공한다.
