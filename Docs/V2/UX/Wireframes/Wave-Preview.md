## 1. 화면 개요

낮 종료 버튼을 누른 후 나타나는 전용 풀스크린. 이번 밤의 적 웨이브 정보를 확인하고, 분대 초기 배치를 결정하며, 전투를 시작하는 전환점이다.

**화면 유형**: 전용 풀스크린 (모달)
**주요 행동**: 적 정보 확인 / 전투 준비 버튼 클릭
**이전 화면**: 낮 페이즈 허브 (낮 종료 버튼)
**다음 화면**: 밤 페이즈 HUD (일시정지 상태로 진입)

## 5. 정보 요소

| 요소 | 위치 | 우선순위 | 조건 |
|---|---|---|---|
| 진입 방향 화살표 | 지도 위 | 최고 | 항상 |
| 규모 텍스트 (소/중/대) | 화살표 옆 | 최고 | 항상 |
| 주공/양동 구분 | 화살표 색상 코딩 | 높음 | 항상 (방향이 여러 개일 때) |
| 적 아키타입 아이콘 | 적 구성 패널 | 높음 | 탐사 Lv1+ 또는 감시탑 Lv2+ |
| 적 종류별 수량 | 아이콘 옆 숫자 | 중간 | 탐사 Lv2+ 또는 감시탑 Lv3 |
| 추가 경로 표시 | 지도 위 화살표 | 중간 | 탐사 Lv2+ 또는 감시탑 Lv3 |
| 이전 밤 대비 비교 | 비교 패널 | 낮음 | 항상 (2일차~) |

### 5.1 화살표 색상 코딩

| 색상 | 의미 |
|---|---|
| 빨강 (진한) | 주공 방향 |
| 주황 | 양동 방향 |
| 화살표 크기 | 규모에 비례 (대규모 = 굵은 화살표) |

## 7. 설계 의도

이 화면은 낮과 밤 사이의 심리적 완충이다. 플레이어에게 두 가지를 제공한다.

첫째, 정보 확인의 기회. 어떤 적이 어디서 오는지를 파악하고 분대 배치 계획을 세울 수 있다. 낮의 건설이 "올바른 방향"이었는지를 확인하는 첫 번째 피드백 지점이다.

둘째, 심리적 준비. 전투 준비 완료 버튼을 누르는 행위 자체가 "나는 준비됐다"는 플레이어의 능동적 선언이다. 자동으로 전투가 시작되지 않는다. 이는 Rising Demacia의 세션 자유 원칙에서 차용한 패턴이다.
