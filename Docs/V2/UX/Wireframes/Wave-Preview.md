## 1. 화면 개요

낮 종료 버튼을 누른 후 나타나는 전용 풀스크린. 이번 밤의 적 웨이브 정보를 확인하고, 분대 초기 배치를 결정하며, 전투를 시작하는 전환점이다.

**화면 유형**: 전용 풀스크린 (모달)
**주요 행동**: 적 정보 확인 / 전투 준비 버튼 클릭
**이전 화면**: 낮 페이즈 허브 (낮 종료 버튼)
**다음 화면**: 밤 페이즈 HUD (일시정지 상태로 진입)

## 5. 정보 요소

| 요소 | 위치 | 우선순위 | 조건 |

| 진입 방향 화살표 | 지도 위 | 최고 | 항상 |

## 6. 인터랙션

| 입력 | 동작 | 피드백 |
|---|---|---|
| [전투 준비 완료] 버튼 | 밤 페이즈 HUD로 전환 (일시정지 상태) | 화면 전환 |
| [분대 초기 위치 확인] 버튼 | 분대 배치 오버레이 열기 (하단 상세 참조) | 지도 위에 분대 위치 오버레이 |
| 화살표 마우스오버 | 해당 방향 상세 정보 툴팁 | 툴팁 |
| 적 아이콘 마우스오버 | 아키타입 이름 + 특성 설명 툴팁 | 툴팁 |

### 6.1 분대 초기 배치 인터랙션 (WaveDefense.md 2.1 "전투 준비" 단계)

[분대 초기 위치 확인] 버튼 클릭 시 기지 지도 위에 분대 배치 오버레이가 활성화된다.

```
+==================================================================+
|                     [ 밤이 다가옵니다 ]                           |
+==================================================================+
|                                                                  |
|  [분대 배치 모드]  -- 분대를 클릭하고 배치 위치를 클릭하세요        |
|                                                                  |
|                      [기지 지도 뷰]                               |
|                                                                  |
|         북쪽                                                      |
|          ↑         [분대3] (현재 위치: 북문)                      |
|          |                                                       |
|  [분대2]<--|--[기지]  ↙ (대규모)                                  |
|     (서문) |                                                      |
|          ↓  [분대1] (현재 위치: 중앙)                             |
|         남쪽                                                      |
|                                                                  |
|  +--- 분대 목록 ---+                                              |
|  | [1] 분대1 — 중앙 |    [클릭: 이동할 분대 선택]                  |
|  | [2] 분대2 — 서문 |                                              |
|  | [3] 분대3 — 북문 |                                              |
|  | [4] 분대4 — 대기 |                                              |
|  +------------------+                                            |
|                                                                  |
+------------------------------------------------------------------+
|  [배치 완료 / 닫기]                      [전투 준비 완료 ->]       |
+==================================================================+
```

**배치 방식**: 클릭 기반 위치 선택. 드래그 없음.

| 단계 | 입력 | 동작 | 피드백 |
|---|---|---|---|
| 1. 분대 선택 | 분대 목록 클릭 또는 숫자 1~5 | 해당 분대 선택 | 분대 아이콘 강조 |
| 2. 위치 선택 | 지도 위 배치 가능 위치 클릭 | 선택 분대 해당 위치로 이동 예약 | 위치 마커 표시 |
| 3. 확인 | [배치 완료] 또는 재클릭 | 배치 저장 | 분대 목록 위치 텍스트 업데이트 |

**배치 가능 위치**: 방어 거점(문, 방벽 후방, 중앙 광장 등) 미리 정의된 주둔 포인트. 자유 위치 이동이 아닌 거점 선택 방식이다. 이는 밤 페이즈 HUD의 자유 이동 명령(우클릭)과 구별된다.

**전투 준비 완료 버튼과의 관계**: 분대 배치 확인 없이 [전투 준비 완료]를 눌러도 된다. 배치를 변경하지 않으면 전투 결과 화면에서 설정된 기본 위치(중앙)를 유지한다.

## 7. 설계 의도

이 화면은 낮과 밤 사이의 심리적 완충이다. 플레이어에게 두 가지를 제공한다.

첫째, 정보 확인의 기회. 어떤 적이 어디서 오는지를 파악하고 분대 배치 계획을 세울 수 있다. 낮의 건설이 "올바른 방향"이었는지를 확인하는 첫 번째 피드백 지점이다.

둘째, 심리적 준비. 전투 준비 완료 버튼을 누르는 행위 자체가 "나는 준비됐다"는 플레이어의 능동적 선언이다. 자동으로 전투가 시작되지 않는다. 이는 Rising Demacia의 세션 자유 원칙에서 차용한 패턴이다.
