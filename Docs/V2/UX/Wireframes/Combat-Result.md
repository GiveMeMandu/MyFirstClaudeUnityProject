## 1. 화면 개요

밤 전투가 끝난 직후 표시되는 전용 화면. 이번 밤의 결과를 정리하고, 다음 낮에 무엇을 해야 하는지 방향을 잡아준다. 뿌듯함(완벽 방어) 또는 긴장감(피해 발생)의 감정적 결말을 명확히 전달한다.

**화면 유형**: 전용 풀스크린
**주요 행동**: 결과 확인 + 계속 버튼
**이전 화면**: 밤 페이즈 HUD (마지막 적 처치 후 자동 전환)
**다음 화면**: 낮 페이즈 허브 (다음 턴)


## 5. 정보 요소

| 요소 | 표시 내용 | 우선순위 | 조건 |
|---|---|---|---|
| 판정 배너 | "완벽 방어!" / "방어 성공" / "경미 피해" / "대규모 피해" | 최고 | 항상 |
| 피해 요약 | 손상 건물 수 + 부상 시민 수 + HQ HP 변화 | 최고 | 항상 |
| 보상 목록 | 기초/고급/유물 획득량 + 잔해 수거량 | 높음 | 항상 |
| 수리 필요 목록 | 손상 건물 이름 + 수리비 (우선순위 순) | 높음 | 손상 건물 있을 때만 |
| 부상 시민 정보 | 시민 이름 + 회복 예상 턴 | 높음 | 부상자 있을 때만 |
| 다음 웨이브 예고 | 없음 (웨이브 미리보기에서 제공) | — | 표시 안 함 |

### 5.1 판정 기준

| 판정 | 기준 |
|---|---|
| 완벽 방어 | 건물 손상 0 + HQ 피해 0 |
| 방어 성공 (경미) | 건물 손상 1~2개 또는 HQ 피해 10% 미만 |
| 방어 성공 (중간) | 건물 손상 3~4개 또는 HQ 피해 10~25% |
| 대규모 피해 | 건물 손상 5개+ 또는 HQ 피해 25%+ |

## 6. 인터랙션

| 입력 | 동작 | 피드백 |
|---|---|---|
| [다음 낮으로 계속] 버튼 | 낮 페이즈 허브로 전환 | 화면 전환 페이드 |
| 손상 건물 항목 클릭 | 건물 상세 툴팁 | 수리 효과 + 수리비 확인 |
| 부상 시민 항목 클릭 | 시민 상태 툴팁 | 회복 일정 + 상태 확인 |
| Escape / Enter | [다음 낮으로 계속]과 동일 | 화면 전환 |

**건너뛰기 없음**: 이 화면은 건너뛸 수 없다. 전투 결과는 플레이어가 반드시 확인해야 하는 핵심 피드백 지점이다. 결과를 무시하고 바로 낮으로 넘어가는 옵션은 제공하지 않는다.

## 7. 시각 연출 방향

| 판정 | 연출 |
|---|---|
| 완벽 방어 | 골드 배너 + 파티클 이펙트 + 팡파레 사운드 |
| 방어 성공 | 흰색/청색 배너 + 완료 사운드 |
| 대규모 피해 | 적색 배너 + 경고 사운드 + 화면 테두리 잠깐 적색 |

배너는 0.5초 지연 후 등장한다. 즉시 표시가 아닌 짧은 지연이 감정적 타이밍을 만든다.
