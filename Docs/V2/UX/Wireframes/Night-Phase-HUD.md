## 1. 화면 개요

밤 전투의 실시간 플레이 화면. 타워디펜스 자동 전투가 기본이고, 플레이어는 분대를 조작하는 가벼운 RTS 마이크로를 수행한다. 일시정지(Space)로 언제든 멈추고 명령을 내릴 수 있다.

**화면 유형**: 영구 HUD (전투 중 지속)
**주요 행동**: 분대 이동/공격/정지 명령 / 일시정지 / 배속 조절
**목표 APM**: 평균 10~20 APM (SC2의 1/10 수준)

## 3. 정보 요소별 상세

### 3.1 상단 HUD

```
+------------------------------------------------------------------+
|  [웨이브: 2/3]  [남은 적: 47]           [HQ HP: ===== 75%]      |
+------------------------------------------------------------------+
```

| 요소 | 위치 | 표시 내용 | 갱신 빈도 |
|---|---|---|---|
| 웨이브 진행 | 상단 좌측 | 현재 서브웨이브 / 총 서브웨이브 | 서브웨이브 전환 시 |
| 남은 적 수 | 상단 중앙 좌측 | 스폰된 적 중 아직 살아있는 수 | 적 처치 시 즉시 |
| HQ HP | 상단 우측 | HP 바 + 퍼센트 | 피해 발생 시 즉시 |

**HQ HP 경고** (Day-Phase-Hub.md 3.2와 동일한 경계값 기준):
- 51% 이상: 녹색 (일반 상태)
- 50% 이하: 황색 전환 + 바 깜빡임
- 25% 이하: 주황 + 빠른 깜빡임
- 10% 이하: 적색 + 경보 사운드

### 3.2 분대 패널 (좌측)

```
+----------+
|  분대 1  |
|  HP: === |  (녹색)
|  이동 중  |
+----------+
|  분대 2  |
|  HP: ==  |  (황색, 피해 50%)
|  정지    |
+----------+
|  분대 3  |
|  HP: ==== |  (녹색)
|  공격    |
+----------+
|  분대 4  |
|  HP: === |  (녹색)
|  대기    |
+----------+
```

| 요소 | 표시 내용 | 갱신 빈도 |
|---|---|---|
| 분대 아이콘/번호 | 분대 식별 (1~5) | 고정 |
| HP 바 | 현재 HP 비율 | 피해 발생 시 즉시 |
| 현재 명령 | 이동 / 공격 / 정지 / 대기 텍스트 | 명령 변경 시 즉시 |
| 선택 강조 | 선택된 분대 테두리 강조 | 선택 변경 시 즉시 |

**HP 색상 기준**:
- 80% 이상: 녹색
- 50~79%: 황색
- 50% 미만: 적색

### 3.3 건물 HP 오버레이

전장 뷰에서 방어 건물 위에 직접 표시.

| 요소 | 표시 위치 | 표시 내용 | 갱신 빈도 |
|---|---|---|---|
| 건물 HP 바 | 건물 위 (월드 공간) | HP 비율 바 | 피해 발생 시 즉시 |
| 손상 경고 아이콘 | HP 바 옆 | 50% 이하 시 경고 삼각형 | 손상 시 즉시 |

### 3.4 미니맵 (우측 하단)

```
+--------+
| 미니맵  |
|        |
| [기지]  |
| * * *  |  (* = 적)
| [분대]  |
+--------+
```

| 요소 | 표시 내용 |
|---|---|
| 기지 구역 | 기지 영역 윤곽 |
| 적 위치 | 적색 점 (밀집도에 따라 크기 가변) |
| 분대 위치 | 청색 삼각형 |
| HQ 위치 | 특수 아이콘 (항시 표시) |

클릭 시 해당 위치로 전장 뷰 스크롤.

### 3.5 하단 컨트롤 바

```
+------------------------------------------------------------------+
|  [1x | 2x]   [|| 일시정지]               [!!] 감시탑 손상!        |
+------------------------------------------------------------------+
```

| 요소 | 위치 | 표시/기능 |
|---|---|---|
| 배속 토글 | 하단 좌측 | 1x / 2x 토글. 현재 배속 강조 |
| 일시정지 버튼 | 하단 중앙 좌측 | Space와 동일. 일시정지 중 "일시정지" 텍스트 강조 |
| 이벤트 알림 | 하단 우측 | 건물 손상 등 비블로킹 경고. 3초 자동 소멸 |

**2x 배속 중 건물 손상 자동 처리**: 손상 발생 시 자동으로 1x로 전환 + "배속 해제: 건물 손상" 알림. 설정에서 비활성화 가능.

### 3.6 현재 명령 표시 (하단 중앙)

```
[명령: 이동]   (분대 선택 후 A키 입력 시: [명령: 공격이동])
```

분대 선택 후 명령 키 입력 시 현재 대기 중인 명령 유형을 표시. 지점 클릭으로 명령 확정.

## 4. 인터랙션 패턴

| 입력 | 조건 | 동작 | 피드백 |
|---|---|---|---|
| 좌클릭 (분대 아이콘) | 언제든 | 해당 분대 선택 | 분대 패널 선택 강조 + 전장 뷰 분대 강조 |
| 좌클릭 (전장 내 분대) | 언제든 | 해당 분대 선택 | 동일 |
| 숫자 1~5 | 언제든 | 분대 선택 | 동일 |
| 우클릭 (빈 지점) | 분대 선택 후 | 이동 명령 | 이동 목표 아이콘 + 분대 이동 시작 |
| 우클릭 (적 대상) | 분대 선택 후 | 공격 명령 | 공격 목표 강조 |
| A 키 + 지점 클릭 | 분대 선택 후 | 공격이동 명령 | 경로상 경고 지시선 |
| H 키 | 분대 선택 후 | 정지 명령 | 분대 정지 + "정지" 상태 표시 |
| Space | 언제든 | 일시정지 토글 | 화면 가장자리 청색 테두리 + "일시정지" 오버레이 텍스트 |
| 배속 버튼 | 언제든 | 1x ↔ 2x 전환 | 버튼 강조 전환 |
| 미니맵 클릭 | 언제든 | 전장 뷰 스크롤 | 뷰 이동 |

## 5. 일시정지 상태 표시

```
+==================================================================+
|  [웨이브: 2/3]  [남은 적: 47]          [HQ HP: ===== 75%]       |
+==================================================================+
|  P A U S E D                                                     |  <-- 반투명 오버레이
|  +-- 분대 --+                                                    |
|  | [1] ===  |                                                    |
|  | [2] ==   |                   [전장 뷰 — 정지]                  |
|  ...                                                             |
|                                                          +------+|
|                                                          |미니맵||
|                                                          +------+|
|  [명령: 없음]                                                    |
+------------------------------------------------------------------+
|  [1x | 2x]   [> 재개]                  (알림 없음)               |
+==================================================================+
```

일시정지 중에도 분대 명령 입력이 가능하다. 명령은 재개 즉시 실행된다.

## 6. 분대 손실 처리

분대원 전원 탈락(HP 0) 시:

| 상황 | UX 처리 |
|---|---|
| 분대 탈락 | 분대 패널에서 해당 분대 회색 처리 + "탈락" 텍스트 |
| 탈락 후 명령 시도 | "분대가 없습니다" 툴팁 |

분대가 0개가 된 상태에서도 전투는 계속된다 (타워 자동 방어).

## 7. 전투 종료 처리

마지막 적 처치 또는 HQ HP 0 도달 시:

| 결과 | 처리 |
|---|---|
| 웨이브 성공 | "웨이브 N 완료!" 배너 (1초) → 다음 서브웨이브 또는 전투 종료 |
| 전투 종료 (생존) | 전투 결과 화면(SCR-06)으로 자동 전환 |
| HQ 파괴 (게임오버) | "본부 파괴" 풀스크린 연출 → 게임오버 화면(SCR-07)으로 전환 |
