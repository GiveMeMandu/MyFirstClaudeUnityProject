## 1. 화면 개요

낮 페이즈의 중심 허브. 건설 탭, 관리 탭, 탐험 탭을 전환하며 기지를 경영한다. 탭은 점진적으로 해금된다. 항상 표시되는 상단 HUD가 자원과 본부 HP를 제공한다.

**화면 유형**: 탭 허브 (영구)
**주요 행동**: 건설 결정 / 인력 배치 / 탐사 파견 / 턴 종료

## 3. 상단 HUD (공통, 항시 표시)

```
+==================================================================+
|  [기초: 48  +8/T]  [고급: 12  +3/T]  [유물: 1]    [HQ HP: ===75%]  |
+==================================================================+
```

### 3.1 정보 요소

| 영역 | 표시 정보 | 우선순위 | 갱신 빈도 |
|---|---|---|---|
| 기초 자원 | 현재 잔고 + 턴당 수입 (+N/T) | 최고 | 매 턴 시작 + 소비 시 즉시 |
| 고급 자원 | 현재 잔고 + 턴당 수입 (+N/T) | 최고 | 매 턴 시작 + 소비 시 즉시 |
| 유물 자원 | 현재 잔고만 (수입 없음) | 높음 | 획득/소비 시 즉시 |
| HQ HP | HP 바 + 퍼센트 텍스트 | 최고 | 밤 전투 결과 반영 시 |

### 3.2 HQ HP 경고 상태

| HP 구간 | 색상 | 추가 표현 |
|---|---|---|
| 100% ~ 51% (51% 이상) | 녹색 | 일반 상태 |
| 50% ~ 26% (50% 이하) | 황색 | 바 깜빡임 + "본부 손상 주의" 툴팁 |
| 25% ~ 11% | 주황 | 바 빠른 깜빡임 + 경고 아이콘 |
| 10% 이하 | 적색 | 바 강한 깜빡임 + 경고 사운드 트리거 |

> **경계값 규칙**: HP 정확히 50%는 황색. HP 51% 이상은 녹색. Night-Phase-HUD.md와 동일한 기준을 적용한다.

### 3.3 자원 부족 경고

| 상황 | UX 처리 |
|---|---|
| 잔고 0 도달 | 해당 자원 텍스트 적색 전환 |
| 건설 비용 부족 | 해당 슬롯/버튼 회색 처리 + "자원 부족" 툴팁 |

## 5. 건설 탭 (SCR-03A)

```
+==================================================================+
|  [기초: 48  +8/T]  [고급: 12  +3/T]  [유물: 1]    [HQ HP: ===75%]  |
|  [1] 건설*   [2] 관리    [3] 탐험       [연구]      [낮 종료 >>]  |
+==================================================================+
|                                                                  |
|                    [기지 뷰 — 등각 투영]                           |
|                                                                  |
|    [슬롯 A: 채집장]    [슬롯 B: 채집장]    [슬롯 C: 미해금]         |
|    [건설 완료]         [건설 가능]         [???]                   |
|                                                                  |
|    [슬롯 D: 감시탑]    [슬롯 E: 방벽]      [슬롯 F: 정제소]         |
|    [건설 완료]         [건설 중 2T]        [건설 가능]             |
|                                                                  |
|                     [HQ: 본부]                                   |
|                     [특수 건물]                                   |
|                                                                  |
+------------------------------------------------------------------+
|  [현재 턴: 3일차]    [원정대: 파견 없음]                           |
+==================================================================+
```

### 5.1 슬롯 상태별 표시

| 슬롯 상태 | 시각 표현 | 클릭 시 동작 |
|---|---|---|
| 잠금 (Locked) | 안개 처리 또는 물음표 + 해금 조건 | 해금 조건 툴팁만 표시 |
| 해금 (건설 전) | 비활성 외형 (반투명 건물 실루엣) | 건설 패널 열기 |
| 건설 중 | 공사 이미지 + "N턴 후 완공" 텍스트 | 건설 정보 표시 (취소 불가) |
| 건설 완료 | 완성 건물 모델 + 소켓 표시 | 업그레이드 패널 열기 |
| 손상 | 건물 위 붉은 경고 아이콘 + HP 바 | 수리 패널 열기 |

### 5.1-R 수리 패널 (PNL-01R) 오버레이 — 손상 슬롯 클릭 시

건설 탭에서 손상(Damaged) 상태 슬롯을 클릭하면 수리 패널이 기지 뷰 우측에 오버레이로 등장한다. 기지 뷰를 가리지 않는다.

```
                             +---------------------------+
                             |  [슬롯 D: 감시탑] — 손상  |
                             |  ========================= |
                             |  현재 HP: 40% (손상 상태) |
                             |                           |
                             |  수리 비용:               |
                             |    기초 자원 5            |
                             |                           |
                             |  수리 후 효과:            |
                             |    HP 100% 복구           |
                             |    소켓 보너스 재활성화    |
                             |                           |
                             |  [수리하기]  [닫기]        |
                             +---------------------------+
```

| 요소 | 표시 내용 | 조건 |
|---|---|---|
| 건물명 + 손상 뱃지 | 슬롯 이름 + "— 손상" 표시 | 항상 |
| 현재 HP | HP % 표시 | 항상 |
| 수리 비용 | 기초 자원 수량 (Construction.md 수리 비용 테이블 참조) | 항상 |
| 수리 후 효과 | HP 100% 복구 + 소켓 보너스 재활성화 설명 | 항상 |
| [수리하기] 버튼 | 자원 차감 후 즉시 수리 실행 | 자원 충분 시 활성 |
| [수리하기] 비활성 | 회색 처리 + "자원 부족" 툴팁 | 기초 자원 부족 시 |

**인터랙션**:

| 입력 | 동작 | 피드백 |
|---|---|---|
| [수리하기] 버튼 클릭 | 기초 자원 차감 + 건물 HP 100% 복구 + 슬롯 상태 Damaged → Active 전환 | 수리 완료 파티클 + 패널 자동 닫힘 |
| [닫기] 버튼 / Escape | 패널 닫기 (수리 취소) | 패널 닫힘 |
| 패널 외부 클릭 | 패널 닫기 | 패널 닫힘 |

**수리 UX 원칙**:
- 수리는 즉시 실행된다 (건설처럼 N턴 대기 없음).
- 수리 확인 대화상자는 없다. 수리 패널 자체가 최종 확인 단계이다.
- 기초 자원 부족 시 버튼만 비활성화되고 패널은 열린 채 유지된다 — 자원 상황을 파악한 뒤 닫을 수 있다.
- 전투 결과 화면에서 수리 필요 건물 항목을 클릭하면 다음 낮 페이즈의 해당 슬롯으로 카메라가 이동하고 수리 패널이 자동으로 열린다.

### 5.2 건설/업그레이드 패널 (PNL-01) 오버레이

슬롯 클릭 시 기지 뷰 우측에 오버레이 패널로 등장. 기지 뷰를 가리지 않는다.

```
                             +---------------------------+
                             |  [슬롯 B: 채집장]          |
                             |  ========================= |
                             |  기본 생산: 기초 +8/턴     |
                             |  소켓 보너스: 건설 적성 시  |
                             |    생산 +15%              |
                             |                           |
                             |  건설 비용:                |
                             |    기초 10  고급 2         |
                             |                           |
                             |  [건설하기]  [닫기]         |
                             +---------------------------+
```

업그레이드 분기 진입 시 분기 비교 팝업(PNL-02)으로 전환.

```
+--------------------------------------------------+
|  업그레이드 방향 선택                              |
|                                                  |
|  +-------------------+  +-------------------+    |
|  |  A: 대형 채집장    |  |  B: 재활용소       |    |
|  |                   |  |                   |    |
|  |  기초 생산 +50%   |  |  기초 생산 +20%   |    |
|  |                   |  |  잔해 수거 자동화  |    |
|  |  비용:            |  |  비용:            |    |
|  |  기초 20  고급 8  |  |  기초 20  고급 8  |    |
|  |                   |  |                   |    |
|  |  [A 선택]         |  |  [B 선택]         |    |
|  +-------------------+  +-------------------+    |
|                                                  |
|  [!] 선택 후 변경 불가                            |
|                                          [닫기]   |
+--------------------------------------------------+
```

### 6.1 정보 요소

| 영역 | 표시 정보 | 우선순위 | 갱신 빈도 |
|---|---|---|---|
| 시민 카드 | 이름 / 적성 / 숙련도 레벨 / 패시브 / 현재 배치 | 높음 | 배치 변경 시 즉시 |
| 소켓 배치 현황 | 건물별 배치 시민 + 숙련도 | 높음 | 배치 변경 시 즉시 |
| 분대 현황 | 분대별 구성 시민 | 높음 | 배치 변경 시 즉시 |
| 미배치 목록 | 현재 배치 없는 시민 목록 | 중간 | 배치 변경 시 즉시 |

### 6.2 인터랙션

| 입력 | 동작 | 피드백 |
|---|---|---|
| 시민 카드 클릭 | 시민 상세 팝업(PNL-03) 열기 | 카드 하이라이트 |
| 시민 카드 드래그 | 배치 위치 변경 | 유효 배치 슬롯 하이라이트, 드롭 시 재배치 |
| 소켓 슬롯 클릭 | 해당 슬롯에 배치된 시민 상세 표시, 또는 미배치 시민 선택 드롭다운 | 슬롯 하이라이트 |
| 배치된 시민에서 다른 슬롯으로 드래그 | 재배치 + 숙련도 페널티 경고 표시 | "재배치 시 숙련도 -1레벨" 경고 툴팁 |

## 7. 탐험 탭 (SCR-03C)

탐험 조건 충족 후 해금. Frostpunk 노드 그래프 UI 차용.

```
+==================================================================+
|  [기초: 48  +8/T]  [고급: 12  +3/T]  [유물: 1]    [HQ HP: ===75%]  |
|  [1] 건설    [2] 관리    [3] 탐험*      [연구]      [낮 종료 >>]  |
+==================================================================+
|                                                                  |
|  +----- 탐사 지도 (노드 그래프) -----+  +-- 원정대 현황 -------+  |
|  |                                    |  |                      |  |
|  |  (노드A)----(노드D)               |  | 1팀: 이동 중          |  |
|  |     |           |                 |  |   시민B, 시민C        |  |
|  |  [기지]---(노드B)---(노드E)[유물] |  |   목표: 노드E         |  |
|  |     |           |                 |  |   남은 턴: 1          |  |
|  |  (노드C)---(노드F)[적접근]        |  |                      |  |
|  |              [!!!]                |  | 2팀: 없음             |  |
|  |                                   |  |   [새 원정 파견]      |  |
|  +------------------------------------+  +----------------------+  |
|                                                                  |
|  [선택: 노드E — "고대 유적"]                                      |
|  유물 노드 | 위험 | 거리 3 | 예상 소요: 3턴                        |
|  [원정대 편성 및 파견]                    [취소]                  |
|                                                                  |
+------------------------------------------------------------------+
|  [현재 턴: 3일차]    [원정대: 파견 중 1팀 (1턴 후 귀환)]           |
+==================================================================+
```

### 7.1 정보 요소

| 영역 | 표시 정보 | 우선순위 | 갱신 빈도 |
|---|---|---|---|
| 탐사 지도 | 노드 그래프 + 상태 아이콘 | 높음 | 탐사 완료 / 원정대 이동 시 |
| 원정대 현황 | 팀 구성 / 목표 노드 / 남은 턴 | 높음 | 매 턴 |
| 선택 노드 정보 | 유형 / 위험도 / 거리 / 예상 소요 | 높음 | 노드 클릭 시 즉시 |
| 적 웨이브 경고 | 적 접근 방향 + "N턴 후 웨이브" | 최고 | 매 턴 업데이트 |

### 7.2 노드 시각적 상태

| 노드 상태 | 시각 표현 |
|---|---|
| 미발견 | 안개 처리 (비가시) |
| 활성화 (미탐사) | 유형 아이콘 + 위험도 색상 테두리 (녹/황/적) |
| 탐사 완료 | 흐린 처리 + 체크 아이콘 |
| 원정대 이동 중 | 점선 경로 + 원정대 아이콘 이동 |
| 적 웨이브 노드 | 붉은 해골 아이콘 + 접근 방향 화살표 |

### 7.3 원정대 편성 팝업 (PNL-04)

```
+----------------------------------------------+
|  원정대 편성 — 목표: 노드E "고대 유적"         |
|                                              |
|  가용 시민:                                   |
|  +-----------+  +-----------+  +-----------+ |
|  | 시민D      |  | 시민E      |  | 시민A      | |
|  | 탐사 Lv0  |  | 범용 Lv0  |  | 전투 Lv2  | |
|  | 패시브:    |  | 패시브:    |  | 현재:      | |
|  | 탐험가직감 |  | 물자조달자 |  | 분대1      | |
|  |            |  |            |  |            | |
|  | [추가]     |  | [추가]     |  | [추가]     | |
|  +-----------+  +-----------+  +-----------+ |
|                                              |
|  원정대 구성 (1/3명):                         |
|  [시민D] [비어있음] [비어있음]                  |
|                                              |
|  [!] 시민A를 추가하면 분대1에서 제외           |
|      분대1 전투력 감소 예상                    |
|                                              |
|  [파견 확인]                    [취소]         |
+----------------------------------------------+
```

**핵심 UX 원칙**:
- 시민 선택 전 영향 경고를 즉시 표시한다. 선택 후가 아니라 시민 카드 위에 마우스를 올리는 순간 영향 미리보기를 표시한다.
- 탐사 적성 시민 카드에는 탐사 아이콘 강조 표시.
- 부상 중 시민은 회색 처리 + 선택 불가.

## 9. 인터랙션 패턴 요약

| 입력 | 동작 | 피드백 |
|---|---|---|
| 슬롯 클릭 | 건설/업그레이드 패널 열기 | 슬롯 하이라이트 |
