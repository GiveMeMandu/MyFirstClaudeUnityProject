2-1. 팝업 _ 최대  생성  가능  개수
한 화면에  생성할  수  있는  팝업  UI 최대  생성  가능  개수를  [Config : 팝업 _ 최대  생성  개수 ] 값으로  관리합니다 .
팝업 생성  가능  조건현재 화면에  생성된  팝업  개수  < [Config : 팝업 _ 최대  생성  개
수]값  이하
[Config : 팝업 _ 최대  생성
개수]값10
*예외  : [팝업  생성  가능  조건 ] 을  만족하지  못했을  때 , 팝업  생성  기능이  실행되었을  경우
추가 팝업을  생성하지  않습니다 . ( 팝업  비정상적으로  많이  생성되는  버그  : 로그  필요 )
 2-2. 팝업 _ 레이어  시스템

개요
팝업의  구성  요소들을  설정할  수  있는  팝업  속성  시스템입니다 .
시스템  목표  : UI 작업자가  프리팹  제작  단계에서  아래  속성들을  설정하여
개발자와  UI 작업자간의  커뮤니케이션  비용을  줄입니다 .
자주 쓰는  팝업  형태를  프리셋  형태로  묶어  개발  효율을  향상시킵니다 .
휴먼 에러를  방지합니다 .

2팝업_닫기  액션  타입 팝업 상단의  닫기  버튼  제공  여부를  결정하는  속성입니다 .
