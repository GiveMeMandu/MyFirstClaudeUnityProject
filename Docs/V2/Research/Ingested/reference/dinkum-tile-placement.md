2. 배치  시스템
2.1. 직접  배치  방식  정의
캐릭터가  건물 / 오브젝트 ( 건물  포함 ) 를  든  상태에서  캐릭터를  직접  움직이며  원하는  위치에  배치하는 방식을  의미
원작의  오브젝트  배치  방식 ( 캐릭터  기반 ) 과  동일하며, 건물  배치  방식 ( 마우스  기반  드래그 & 드롭 ) 과  다름
배치 또는  재배치  시  건물 / 오브젝트  활성화  방식
배치할  건물 / 오브젝트를  툴벨트에  등록 후 해당  아이템  슬롯  선택  시  배치  가능
인벤토리에서 배치할  건물 / 오브젝트  아이템  선택  후  배치하기  버튼 선택  시  배치  가능  ( 임시  툴벨트  기능 )
배치된  오브젝트의  근접  상호작용  범위  내에서  재배치  버튼 선택  시  재배치  가능  ( 임시  툴벨트  기능 )
배치 상황에서  들고  있는  건물 / 오브젝트의  프리뷰  메시  및  배치  인디케이터 출력  필요
배치할  오브젝트  활성화  시  활성화된  오브젝트  면적 (TileSizeX * TileSizeY) 에  비례하여 카메라  자동  줌아웃  처리 필요
카메라  자동  줌아웃  처리는  배치할  오브젝트의  최초 활성화  시점에만 적용
자동 줌아웃  이후  캐릭터  이동 / 카메라  조작  등으로  프리뷰  메시가  가려지는  상황은  대응하지  않음
자동 줌인  처리는  하지  않음
배치 관련  추가  기능
회전 기능: 배치할  오브젝트를  회전하는  기능
재배치  기능 (new): 배치된  오브젝트의  위치  변경을  위한  기능
재배치  취소  기능 (new): 배치된  오브젝트  재배치  상태에서  취소  후  원래  배치된  자리로  되돌리는  기능  (= undo)
