## 3. 턴별 스냅샷 (핵심 턴)

### 범례

- **수입** = 건물 기본 생산 + 소켓 보너스 + 방어/탐사 보상
- **지출** = 건설 + 수리 + 장비 + 연구 + 집회 등
- **잔고** = 이전 턴 잔고 + 수입 - 지출 (음수 불가)
- 유물은 이산적 이벤트이므로 "누적 보유량"으로 표시
- **부록 상세 수치표(섹션 4)가 단일 권위 소스(single source of truth)**

**플레이어 의사결정**: 기초 잔고 60으로 여유로운 출발. 채집장 2호(기초 15)를 우선 건설하여 기초 생산을 2배로 확보하는 것이 가장 효율적. 대안으로 감시탑(기초 12 + 고급 3)을 먼저 짓는 방어 우선 전략.

**비고**: v0.2와 달리 고정 지출(식량 유지비 -12)이 없으므로 초반이 안정적. 유일한 자동 지출은 장비 소모(고급 -1/전투). 초기 잔고의 투자 순서가 핵심.

## 4. 부록: 상세 수치표 (턴 1~25) -- 단일 권위 소스

> 이 표가 **단일 권위 소스(single source of truth)**이다. 위 스냅샷과 모든 분석은 이 표에서 파생된다.
> "경미 피해" 기본 가정. 턴 20에서 1회 "대규모 피해". **모든 잔고는 0 이상.** 패배 조건: 본부 파괴.
> **고정 지출(유지비)은 없다.** 자원 소비는 플레이어 의사결정에서만 발생.
> 분대 장비: 고급 -1/분대/전투 (대규모 피해 시 x1.5). 유물은 이산적 이벤트로 누적 표시.

### 계산 규칙

1. **잔고 = 이전 턴 잔고 + 수입 - 지출** (음수 불가)
2. 잔고 부족 시: 건설/업그레이드 이연. 수리비 미납 시 건물 비활성
3. 수입: 건물 기본 생산(자동) + 소켓 보너스(숙련도 반영) + 방어/탐사 보상
4. 지출: 건설 + 업그레이드 + 수리 + 분대 장비 + 연구 + 모닥불 집회
5. 소켓 보너스: 적성 일치 Lv0 +10%, Lv1 +15%, Lv2 +30%, Lv3 +50%
6. 모든 수치는 산술 검증 완료 (Python 시뮬레이션으로 25턴 전체 계산)

## 5. 패배 조건: 본부 파괴

### 7.3 잔고 패턴 분석

**기초 자원**: 초반 축적(54~100) -> 중반 소비 시작(71~100) -> 후반 불안(53~72)

- v0.2와 달리 기초 0 도달 턴이 **없다**. 고정 유지비 제거 덕분에 기초 자원이 0에 빠지지 않는다.
- 대신 수리비(T20: 25, T25: 25)가 후반 주요 싱크가 되어 "쌓였다 빠지는" 진동 패턴 형성.
- 기초 캡(160)에 도달하는 턴 없음. T9의 100이 최고치이며, 저장소 건설(T7)로 캡이 160으로 확대된 이후 여유.

**고급 자원**: 초반 급락(20->12) -> 중반 진동(4~23) -> 후반 축적(4~31)

- 고급 자원 최저 4(T19). 이 시점이 "고급 자원 고갈 직전"의 핵심 병목 순간.
- 정제소 업그레이드(T17, +4->+6)가 고급 경제의 전환점.
- T22~25에서 고급이 축적되는 것은 건설/연구 수요 소멸로 장비 소모만 남기 때문.

**유물**: 총 획득 5개 (탐사 4, 방어 보상 1) -> 총 소비 2개 (엘리트 연구 1, 기타 1) -> 잔여 3개
